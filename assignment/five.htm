Introduction to HTML

What is HTML?

HTML (Hypertext Markup Language) is the standard language for creating web pages. It describes the structure of a webpage and consists of a series of elements which tell the browser how to display the content.

History of HTML Versions

HTML 1.0

Released in 1993, the first version of HTML provided basic elements such as text, links, and images.

HTML 2.0

Published in 1995, HTML 2.0 included everything from HTML 1.0 plus new features like forms.

HTML 3.2

Released in 1997, HTML 3.2 added support for tables, applets, and text flow around images.

HTML 4.0

Released in 1999, this version included enhanced support for style sheets and scripting languages.

HTML5

The most recent version, HTML5, was finalized in 2014. It introduced new elements, attributes, and behaviors, and a larger emphasis on APIs for complex web applications.
