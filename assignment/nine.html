Text Formatting Tags in HTML

Bold Text

The bold tag is used to make text bold.

Italic Text

The italic tag is used to italicize text.

Underlined Text

The underline tag is used to underline text.

Superscript Text

The superscript tag is used for superscript text.
For example: E=mc2.

Subscript Text

The subscript tag is used for subscript text.
For example: H2O.
